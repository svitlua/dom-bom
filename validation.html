JS: Form Validation



Validation Criteria

using JS create this form (by document.createElement) and append to body on form submit you need validate below fields

  • age should contains only numbers
  • (pass) "123"
  • (pass) "0"
  • (pass) "008946798576948567948576948756984756"
  • (failed) "test"
  • (failed) "-1"
  • (failed) " 0"
  • (failed) "10e5"
  • username should start from "user_" (*or start with any number of symbol '*' before 'user_'?)
  • (pass) **user_**test
  • (pass) **user_**213kjasd
  • (pass) user_@gmail.com
  • (failed) john
  • (failed) _user_test
  • (failed) testuser_test
  • field date should contain current date (format dd/mm/yyyy)
  • if data is not valid, alert message your data is invalid
